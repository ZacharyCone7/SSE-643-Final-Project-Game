# SSE-643-Final-Project-Game
The following information will proide instructions on how to run Campfire Siege

## How to Run:
1. Ensure you have `npm install` in the root.
2. Open the program in IDE
3. Click the "Go Live" button at the bottom of the screen.
4. Use the mouse, WASD and Space Bar to navigate the project

## Description
Game where you try and survive the night against a horde of zombies with nothing but a baseball bat

### Controls
1. Use the mouse to view and aim at zombies. Use the left click to attack the zombies by swinging the bat
2. Use WASD to move around the campfire to attack or avoid zombies.
3. When you die, press the space bar to restart the game.
